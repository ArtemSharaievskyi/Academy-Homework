Правила игры

Игровое поле разделено на клетки, некоторые из которых заминированы. Для победы вам нужно открыть все клетки, не попадая на мины. В открытых клетках отображаются цифры, каждая цифра — это количество мин в соседних клетках. С помощью этой информации можно определить, в каких клетках содержатся мины. Предполагаемую клетку с миной можно пометить флажком с помощью правой кнопки мыши.

Аккорды

Если цифра окружена правильным количеством флагов, то при нажатии на эту цифру остальные окружающие её клетки откроются автоматически — это называется аккордом. Использование аккордов сокращает количество лишних кликов и повышает эффективность игры.

Режим без угадывания (NG)

В стандартном режиме могут возникать ситуации, когда решение невозможно найти с помощью логики и нужно угадывать. Режим без угадывания (No Guessing) позволяет решать все игры только с помощью логики. Чем выше уровень сложности, тем сложнее логические паттерны.

Советы

Начинайте игру с углов. В углах чаще бывают открытия, это повышает вероятность победы.

Используйте флажки для пометки подозрительных клеток и избегайте случайных кликов.
